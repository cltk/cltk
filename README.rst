Installation
============

For the CLTK's latest pre-release version:

.. code-block:: bash

   $ pip install --pre cltk

Requirements:
   - Python version 3.7 or above
   - A Unix-like OS
   - To install from source, see `Development <https://dev.cltk.org/en/latest/includes/development.html>`_ in the docs.

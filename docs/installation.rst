Install via Pip:

.. code-block:: bash

   $ pip install cltk


For a pre-release version:

.. code-block::

   $ pip install --pre cltk


To install from source, clone the Git repository and run:

.. code-block::

   $ make install


See :doc:`development` for more on configuring your environment for CTLK development.
